Business Event驱动对外部系统WebService的调用

  • Create/Use Invoker Event
  • 选Workflow Administrator WebApplications职责
  • 点击“Business Events”
  • 点击“Create Event”
  • 填写Name，Display Name和Description选择状态为“Enabled”
  • 输入Owner Name和Tag（通常都填FND/FND）

  • 点击“Apply”保存

  • Native Service Invocation Process

  • Search Invoker Business Event.
  • Select to view its subscriptions
  • Click "Create Subscription"
  • Enter local Subscriber System,Phase(<100 for synchronous response).
  • Select Rule Data as "Message".
  • Select Action Type "Invoke Web Service".
  • Click "Next".
  • Follow wizard based WSDL parser UI:
  • Enter WSDL URL and click "Next".
  • Select Service and click "Next".
  • Select Service Port and click "Next".
  • Select Service Operation and click "Next".
  • Use seeded Java Rule Function - oracle.apps.fnd.wf.bes.WebServiceInvokerSubscription or custom extended class.

  • Optionally enter WS Security parameters:WFBES_SOAP_USERNAME,WEBES_SOAP_PASSWORD_MOD,WEBES_SOAP_PASSWORD_KEY as

  • Create Receive Event

  • navigate to "Events" tab in Workflow
